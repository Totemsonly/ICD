# 基于 Python 的答题卡检测系统

这是一个轻量级 OMR（Optical Mark Recognition）示例项目，支持：

- 上传答题卡图片；
- 自定义题目数量与每题选项数；
- 自己输入正确答案（如 `A,B,C,D`）；
- 自动识别作答并计算成绩。

## 1. 安装依赖

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 2. 启动服务

```bash
python app.py
```

默认地址：`http://127.0.0.1:5000`

## 3. 使用说明

1. 上传答题卡图片（建议拍摄清晰、光线均匀）；
2. 输入题目数量，例如 `10`；
3. 输入每题选项数量，例如 `4`（A-D）；
4. 输入正确答案，例如 `A,B,C,D,A,B,C,D,A,B`；
5. 点击“开始识别并评分”。

## 4. 识别逻辑说明

- 系统先尝试检测答题卡外轮廓并做透视矫正；
- 将答题区域按“题目数量 × 选项数量”切分成网格；
- 通过每个格子的黑色像素统计，选出最可能涂黑的选项；
- 与你输入的标准答案进行比对并输出成绩。

> 注意：此版本为通用模板，适用于规则较整齐的答题卡。若你有固定版式（比如特定学校模板），可以继续优化识别精度。
